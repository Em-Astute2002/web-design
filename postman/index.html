What is Postman?

Postman is a collaboration platform for API development. Postman's features simplify each step of
building an API and streamlne collaboration so you can create better APIs-faster.

API client

Quickly and easily send REST, SOAP, and GraphQL requests directly within Postman.

Automated Testing
